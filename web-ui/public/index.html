About ZukaiDocs

ZukaiDocs (図解 = "visual explanation") is a powerful Markdown to PDF converter with live preview and Mermaid diagram support.

Features

  • Live preview with syntax highlighting
  • Multiple themes (GitHub, Professional, Academic, Minimal)
  • Full Mermaid diagram support (Flowcharts, Gantt, Sequence, etc.)
  • Export to PDF with proper formatting
  • Export Mermaid diagrams as PNG images
  • Drag and drop file support

Keyboard Shortcuts

  • F11 - Toggle fullscreen preview
  • Ctrl/Cmd + S - Download PDF
  • Esc - Exit fullscreen
